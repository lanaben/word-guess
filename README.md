### ENV:

To both client and server, add variable PASSWORD to .env.

### INSTALLATION, BUILDING, STARTING THE APP:

#### Server:

```bash
cd server_app
npm i
npm run build
npm run start
```

#### Client:

```bash
cd client_app
npm i
npm run build
npm run start
```

#### Website:

Head to http://localhost:3001 to see games.
